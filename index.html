About this project

This is a mock-up for a motivaton/lifestyle brand. Keeping it simple and related to the outdoors.
This site is s mixture of bootstrap, CSS, grid and flex, JavaScript an a little Photoshop to make some quick logos. Im hoping that this is able to expose my ability to use multiple different methods of styling and show at least some of my understanding of JavaScript.

Call To Action Tiles

These are a couple "Call to action" tiles I made using flex, bootstrap cards, and some css. These tiles have a little bit of movement giving them a subtle, yet eye-catching, flare. I enjoy these tiles because they can be used as buttons, links, or internal anchors.
